# Tailoring Shop Portfolio Website

A responsive business website built for a tailoring shop.

## Features
- Fixed navbar with mobile toggle
- Responsive design for mobile, tablet, and desktop
- Image galleries showcasing tailoring works
- Customer reviews section
- Contact section with WhatsApp and Instagram links

## Tech Stack
- HTML
- CSS
- JavaScript
